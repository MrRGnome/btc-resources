Bitcoin Forks

Every node user themselves defines Bitcoin. When users disagree about what Bitcoin is, it forks. Forks can be dangerous events. You can protect yourself by not transacting during a fork, and selling hostile forks.

Dangerous Forking Software

Strongly discouraged: newer versions of this software WILL fork, almost certainly into an extreme minority with no miner support. You risk your node being attacked, finding yourself on a chain with no miners, or tricked into sending transactions which are replayable. User sovereignty means user sovereignty, so you choose.
